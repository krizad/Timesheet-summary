# Timesheet Summary

A modern, efficient tool for processing and visualizing timesheet data from Excel files. Built with React, TypeScript, and Tailwind CSS.

## üöÄ Features

- **Excel Processing**: Upload standard timesheet Excel files (`.xlsx`) to automatically calculate mandays, total hours, and project distributions.
- **Dashboard**: Visual overview of project statistics with interactive charts.
- **Timeline View**: Detailed timeline visualization of work periods with gap detection and manday counts.
- **Data Table**: Comprehensive table view of project data with sorting and filtering capabilities.
- **Dark Mode**: Fully supported dark mode for comfortable viewing in any environment.
- **Responsive Design**: Optimized for desktops, tablets, and mobile devices.

## üèÅ Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/krizad/Timesheet-summary.git
   cd Timesheet-summary
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally

Start the development server:
```bash
npm run dev
```
The application will be available at `http://localhost:5173`.

### Building for Production

Build the application for production:
```bash
npm run build
```

Preview the production build:
```bash
npm run preview
```
